# 📊 Sales Dashboard باستخدام Streamlit

## 🧠 Overview

This project is an **interactive Sales Dashboard** built using Python.
It allows users to explore and analyze sales data efficiently through dynamic filters and clean visualizations.

The dashboard provides key business insights by displaying **KPIs (Key Performance Indicators)** and interactive charts.

---

## 🚀 Features

* 🔍 Data filtering by:

  * City
  * Customer Type
  * Gender

* 📈 Key Performance Indicators:

  * Total Sales
  * Average Rating
  * Average Sales per Transaction

* 📊 Visualizations:

  * Sales by Product Line
  * Sales by Hour

* ⚡ Fully interactive dashboard built with Streamlit

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas (Data Analysis)
* Plotly (Data Visualization)
* Streamlit (Web Application Framework)
* OpenPyXL (Excel File Handling)

---

## 📂 Dataset

The project uses an Excel dataset:

```
supermarkt_sales.xlsx
```

The dataset includes:

* Invoice data
* Cities
* Customer types
* Total sales
* Ratings
* Time

---

## ▶️ How to Run

### 1. Install Dependencies

```bash
pip install pandas plotly streamlit openpyxl
```

### 2. Run the App

```bash
streamlit run app.py
```
